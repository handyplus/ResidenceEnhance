### 一、简介

> ResidenceEnhance 是一款领地的附属插件

### 一.功能介绍

1. 简单的配置即可禁止领地执行指令

```yaml
# 是否也禁止op true 限制.false不限制
isOp: true

# 是否也禁止领地所有者
isOwner: false

msg: "§a这个领地禁止执行该命令!"

# 限制的领地名字
name:
- "test"
- "fly"

# 领地禁止的命令,无视大小写
command:
- "setHome"
- "fly"
```
